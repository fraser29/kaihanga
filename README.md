# kaihanga

Utilities for 3D printing

"kaihanga" is the MÄori word for maker / builder / creator.


## Installation

```bash
pip install -r requirements.txt
```

ring.py requires: 

```bash
sudo apt install openscad
```


## Usage

ring.py - outer and inner diams and height in mm & Name to stencil:

```bash
python3 ring.py  --name Billy

```
